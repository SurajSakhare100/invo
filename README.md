# Invo - Invoice Management System

A modern invoice management platform that helps businesses manage customers, invoices, transactions, and financial analytics efficiently.

## 🚀 Features

- 🔐 User authentication
- 👥 Customer management
- 🧾 Create and manage invoices
- 💰 Track transactions
- 📊 Business analytics dashboard
- 🔍 Search and filter records
- 📱 Responsive user interface
- ⚡ Fast and scalable backend API

## 🛠️ Tech Stack

### Frontend
- React + Vite
- TypeScript
- Tailwind CSS
- React Router
- Axios

### Backend
- Node.js
- Express.js
- TypeScript
- MongoDB
- Mongoose
- JWT Authentication

### Tools
- Git & GitHub
- REST API
- ESLint
- Prettier
